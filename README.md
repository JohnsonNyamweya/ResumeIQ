Description
This application allows users to sign up, create and manage resumes with live previews, shareable links, and AI-driven optimizations. Whether you're starting from scratch or uploading an existing resume, ResumeIQ uses Google Gemini AI to suggest improvements and optimize your content. Profile images can be uploaded with automatic background removal using ImageKit.
The project is built step-by-step:

Frontend: React.js and Tailwind CSS for a responsive and intuitive UI.
Backend: Node.js with Express.js for handling API requests, and MongoDB for storing resume data.
AI Integration: Google Gemini AI model for resume optimization.
Image Handling: ImageKit for storing and processing profile images.

Features
User Authentication: Secure sign-in and sign-up functionality.
Create New Resume: Start building a resume from scratch with easy-to-use forms.
Live Preview and Online Link: Real-time preview of your resume with a unique shareable URL.
Share Resume: Easily share your resume via live links for job applications or networking.
AI Optimization: Upload an existing resume and let AI suggest improvements for better impact.
Resume Management: Add, delete, and edit resumes with full control.
Image Upload with Background Removal: Upload profile pictures and automatically remove backgrounds for a professional look.
Multiple Templates: Choose from a variety of resume templates to suit different industries and styles.

Tech Stack
Frontend: React.js, Tailwind CSS
Backend: Node.js, Express.js
Database: MongoDB
AI: Google Gemini AI
Image Storage: ImageKit
Other: JWT for authentication (assumed based on common practices), RESTful APIs
